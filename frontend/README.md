# Frontend (Next.js)

UI for tracking repositories and deployments.

## Requirements
1. Node.js 20+

## Environment
1. `NEXT_PUBLIC_API_URL`  
   Example: `http://localhost:8002`

## Run Locally (without Docker)
```bash
cd frontend
npm install
npm run dev
```

## Build
```bash
cd frontend
npm run build
```

## Scripts
1. `npm run dev`
2. `npm run build`
3. `npm run start`
4. `npm run lint`
5. `npm run lint:fix`
6. `npm run format`
7. `npm run format:check`
